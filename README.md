O **Trabalho Jordanny** é um sistema de catraca inteligente desenvolvido para automatizar o controle de presença escolar. Utilizando um cartão de identificação (como RFID), cada aluno registra sua entrada ao passar pela catraca, eliminando a necessidade de chamada manual em sala.

O sistema funciona com um dispositivo eletrônico que lê o cartão e envia os dados para um backend, responsável por registrar automaticamente a presença com base no horário. As entradas são classificadas como **presente** (até 06:42), **atrasado** (até 07:15) ou **falta** (após esse horário).

A proposta torna o processo mais rápido, preciso e organizado, além de permitir o acompanhamento eficiente da frequência dos alunos. O projeto utiliza tecnologias como Python, FastAPI e ESP32, integrando hardware e software para resolver um problema real no ambiente escolar.
